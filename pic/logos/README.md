## 下载建议
- 优先选择 PNG 格式（支持透明背景）
- 如果是 SVG 格式，请转换为 PNG
- 确保 Logo 清晰，适合在网页上展示
- 可以使用学校官网的 Logo 或从维基百科下载

## 备选方案
如果无法下载官方 Logo，可以：
1. 使用学校首字母作为占位符
2. 使用纯色圆形图标
3. 使用校徽简化版本
